# PythonProjects
#### These are some well-detailed projects written in Python for beginners and learners in general, with variety in difficulty levels.
### Easy level:
1. [Dice Rolling](easy/dice)
  using:
    - `random` module.
    - `random.choice()` function.
    - `str.lower()` function.
    - `KeyboardInterrupt` exception.
2. [Guess the number](easy/guess-the-number)
  using:
    - `random` module.
    - `random.randint()` function
    - `str.lower()` function
    - `KeyboardInterrupt` exception.
3. [Hangman](easy/hangman)
  using:
    - Function annotations.
    - Variables annotations.
    - String formatting.
    - `random` module.
    - `random.choice()` function.
    - `str.split()` function.
    - `str.join()` function.
    - `filter()` function.
    - `lambda` expression.
    - `Global` statement.
    - `KeyboardInterrupt` exception.
